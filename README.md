# Art-Portfolio

A modern art portfolio website built with Next.js, React, and Tailwind CSS. Features a responsive design with dark/light theme toggle, animated components using Framer Motion, and sections for gallery, portfolio, commissions, shop, and contact.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
npm start
```
